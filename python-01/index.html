自学Python-01 环境搭建

Author: wangzhengcheng Date: June 28, 2020  10:18:47 Category: Python

一、安装

1. window系统

官网下载地址：https://www.python.org/

安装页面有”add python 3.7 to PATH”,勾选该选项就不需要自己手动再配置环境变量了

验证是否安装成功，打开命令行窗口，输入”python”命令，出现版本号即安装成功

2.mac系统

一般自带2.x版本，建议安装最新版本，官网下载地址:https://www.python.org/downloads/mac-osx/

但我推荐使用Homebrew下载，Homebrew是mac上很强大的包管理工具，没有安装可以将以下命令粘贴至终端进行安装：

安装之后执行命令

brew install python3

输入 python3 查看版本信息，安装完成。

用Homebrew是不是很方便，还有个好处是不用手动配置环境变量，Homebrew会帮我们完成。

二、开发工具

Sublime 、VSCode、PyCharm都不错，个人喜欢用VSCode，或者学习期间用记事本写都行。

三、第一个python

打开文本编辑器，输入 print("hello") 保存文件为hello.py,后缀是py

运行，打开命令行终端到文件根目录下，执行python3 hello.py 运行命令很简单，使用 python3 命令+文件名即可

输出结果为”hello”
